Navigation Fullscreen Block - Demo Variants

Standalone fullscreen overlay navigation triggered by external elements. Copy these snippets into WordPress Code Editor.

How to Trigger This Navigation

Add data-nav-trigger="your-trigger-id" to ANY element on your page (button, link, hamburger menu, etc.) to open this navigation overlay.

Example: <button data-nav-trigger="main-nav">Open Menu</button>

JavaScript API Advanced

Control navigation programmatically via JavaScript.

// Open navigation by trigger ID
window.AlpacodeNavFullscreen.open('main-nav');

// Close navigation
window.AlpacodeNavFullscreen.close('main-nav');

// Toggle navigation
window.AlpacodeNavFullscreen.toggle('main-nav');

// Close all open navigations
window.AlpacodeNavFullscreen.closeAll();

// Listen for events
document.addEventListener('navfullscreen:open', function(e) {
    console.log('Navigation opened');
});

document.addEventListener('navfullscreen:close', function(e) {
    console.log('Navigation closed');
});
