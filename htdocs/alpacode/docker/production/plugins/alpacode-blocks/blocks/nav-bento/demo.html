Navigation Bento Block - Demo Variants

Creative bento-grid style fullscreen navigation. Each tile can have its own color, size, icon, and sub-items.

How to Trigger This Navigation

Add data-nav-bento-trigger="your-trigger-id" to ANY element to open this navigation.

Example: <button data-nav-bento-trigger="bento-nav">Menu</button>

JavaScript API Advanced

Control the bento navigation programmatically.

// Open navigation by trigger ID
window.AlpacodeNavBento.open('bento-nav');

// Close navigation
window.AlpacodeNavBento.close('bento-nav');

// Toggle navigation
window.AlpacodeNavBento.toggle('bento-nav');

// Close all open navigations
window.AlpacodeNavBento.closeAll();

// Listen for events
document.addEventListener('navbento:open', function(e) {
    console.log('Bento navigation opened');
});

document.addEventListener('navbento:close', function(e) {
    console.log('Bento navigation closed');
});

// Access instance directly
var nav = document.querySelector('.alpacode-nav-bento[data-trigger-id="bento-nav"]');
if (nav && nav.alpacodeNavBento) {
    nav.alpacodeNavBento.open();
    console.log('Is open:', nav.alpacodeNavBento.isOpen());
}
